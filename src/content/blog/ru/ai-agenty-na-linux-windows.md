## Вариант 1: VPS-сервер в облаке

**Стоимость: от 800 руб/мес (сервер) + от 2 000 руб/мес (подписка Claude)**

Подходит если не хотите ничего устанавливать на свой компьютер.

![Терминал с Claude Code](/blog/ai-agents-terminal.jpg)

#### Шаг 1: Арендуйте сервер

Рекомендуемый провайдер: **Hetzner** (hetzner.com)
- Тариф CX22 (2 vCPU, 4 GB RAM, 40 GB SSD) - 450 руб/мес
- Или CX32 (4 vCPU, 8 GB RAM, 80 GB SSD) - 800 руб/мес
- ОС при создании: **Ubuntu 24.04**

Альтернативы: Timeweb Cloud, Selectel, DigitalOcean.

#### Шаг 2: Подключитесь к серверу

На Windows: скачайте PuTTY или Windows Terminal. На Mac: откройте Терминал.

```
ssh root@ваш_ip_адрес
```

#### Шаг 3: Установите Claude Code

```
apt update && apt upgrade -y

curl -fsSL https://deb.nodesource.com/setup_22.x | bash -
apt install -y nodejs

npm install -g @anthropic-ai/claude-code

apt install -y ffmpeg python3-pip git
pip3 install --break-system-packages Pillow requests httpx
```

#### Шаг 4: Авторизуйтесь

```
claude
```

Следуйте инструкциям на экране - нужно будет ввести ключ подписки Anthropic.

#### Шаг 5: Работайте

```
claude

claude -p "Придумай 5 идей для постов магазина электроники"
```

## Вариант 2: Свой компьютер с Linux

**Стоимость: 0 руб (если есть ПК) + от 2 000 руб/мес (подписка Claude)**

Подходит если есть старый или новый ПК, готовы поставить Linux вместо Windows.

#### Минимальные требования к компьютеру

| Параметр | Минимум | Рекомендуется |
|----------|---------|---------------|
| Процессор | Intel i3 / AMD Ryzen 3 | Intel i5 / AMD Ryzen 5 |
| Оперативная память | 4 ГБ | 8-16 ГБ |
| Диск | 120 ГБ SSD | 256 ГБ SSD |
| Интернет | Стабильный | Стабильный |
| Видеокарта | Не нужна | Не нужна |

> **Важно:** Видеокарта НЕ нужна. Вся "умная" работа происходит в облаке на серверах Anthropic. Ваш компьютер просто отправляет запросы и получает ответы.

#### Шаг 1: Установите Ubuntu

1. Скачайте Ubuntu 24.04 LTS: ubuntu.com/download/desktop
2. Запишите на флешку (8 ГБ+) программой Rufus (Windows) или balenaEtcher
3. Загрузитесь с флешки (F12 или Del при включении)
4. Выберите "Установить Ubuntu" и следуйте инструкциям
5. Перезагрузитесь

#### Шаг 2: Откройте терминал

Нажмите Ctrl+Alt+T или найдите "Терминал" в меню приложений.

#### Шаг 3: Запустите установку

Скопируйте и вставьте эту команду целиком:

```
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo bash - && \
sudo apt install -y nodejs ffmpeg python3-pip git && \
sudo npm install -g @anthropic-ai/claude-code && \
sudo pip3 install --break-system-packages Pillow requests httpx && \
echo "Готово! Запустите: claude"
```

#### Шаг 4: Авторизуйтесь и работайте

```
claude
```

## Вариант 3: Windows (через WSL2)

**Стоимость: 0 руб + от 2 000 руб/мес (подписка Claude)**

Подходит если не хотите удалять Windows.

#### Шаг 1: Включите WSL2

Откройте PowerShell от администратора и выполните:

```
wsl --install
```

Перезагрузите компьютер. При загрузке создайте логин и пароль для Ubuntu.

#### Шаг 2: Установите Claude Code

В терминале Ubuntu (WSL2):

```
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo bash - && \
sudo apt install -y nodejs && \
sudo npm install -g @anthropic-ai/claude-code && \
echo "Готово! Запустите: claude"
```

#### Шаг 3: Работайте

```
claude
```
